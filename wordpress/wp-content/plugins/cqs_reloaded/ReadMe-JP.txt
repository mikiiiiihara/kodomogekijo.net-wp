▼はじめに （2016/5/26 時点）
Custom Query String Reloaded v2.9 の配布先ページが閉鎖されています。
このWordPressプラグインはどうしても使用を続けたいと思っています。
そこで思い切って日本語表示と最低限の機能追加を行いました。
但しメインロジックには一切の修正をしていません。
愛する（怖い）妻からの要望で作業を行いました。
せっかくなので一般公開することにします。

追記（2017/06/20）
PHP7.0に対応させるためにコンストラクタを__construct()に変更します。
http://php.net/manual/ja/migration70.deprecated.php　より引用。
「PHP 4 形式のコンストラクタ (クラス名と同じ名前のメソッドを定義するもの) が非推奨となりました。将来的に削除される見込みです。 PHP 7 では、クラス内でこの形式のコンストラクタしか定義されていない場合に E_DEPRECATED が発生します。 __construct() メソッドを実装していれば、この警告は発生しません。」


▼内容
Custom Query String Reloaded v2.9 に対して以下の修正と追加を行いました。
・日本語が表示できる為の国際化ロジックを追加
・国際化以外のメインロジックは変更無し
・設定の画面が日本語表示になるための言語ファイルを追加
・PHP7.0に対応させるためにコンストラクタを__construct()に変更
